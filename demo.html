IconFontRater

Basics

A simple, font based, Rating widget. The widget is extremely simple, but can easily be expanded upon.

Customizable

By changing out the font, can get a different effect. I've included a custom icomoon font, and some reasonable defaults.

  • empty - default: '&#xe600' (empty star)
  • half - default: '&#xe601' (half star)
  • full - default: '&#xe602' (full star)

By setting properties can further customize

  • numStars - default: 5 (the number of ratable units)
  • value - default: 0 (the number of stars to be filled in

Scalable

Font based. The "images" scale well.

Interactive

Added an alternate version, which with just a few lines of code have added click handling. Show's that I don't have to lose anything by using iconfonts. The same could be done to provide a hover effect as well if so desired.

Technologies

  • Dojo
    • Dijit - an (awesome) widget framework.
    • parser - use widget declaratively
  • Icomoon IconFonts - custom built and crips icon fonts
